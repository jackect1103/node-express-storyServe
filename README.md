# 基于node的后台
 * 使用Express 应用程序生成器

 **通过应用生成器工具 express-generator 可以快速创建一个应用的骨架。**

## 安装express-generator

```javascipt
express-generator 包含了 express 命令行工具

npm install express-generator -g
```
- [express官网](http://www.expressjs.com.cn/starter/generator.html)

* 使用express创建项目文件
```javascipt
express -e 项目名称
```
* 安装所有依赖包
```javascipt
$ cd myapp
$ npm install
```
